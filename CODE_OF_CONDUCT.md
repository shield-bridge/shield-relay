# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its code of conduct.

The full text is published here:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

In short: be respectful and constructive in every project space, and when representing
the project in public. Exclusionary or abusive behavior toward other participants is not
tolerated.

## Reporting

To report a concern, contact the maintainer privately at `andrewkishino@gmail.com`.
All reports are reviewed and handled confidentially.

Enforcement follows the Contributor Covenant's Enforcement Guidelines (see the link
above).
